# Manual Dependency Injection

A Core Java project demonstrating **Manual Dependency Injection (DI)** and **Loose Coupling** without using the Spring Framework. This project is part of my journey to learn the fundamentals behind Spring before moving on to Spring Boot and Microservices.

---

## 📌 Objective

The goal of this project is to understand how Dependency Injection works internally by manually wiring objects together using Core Java.

Instead of relying on the Spring IoC Container, all dependencies are created and injected manually. This helps build a strong understanding of the concepts that Spring automates.

---

## 🚀 Features

* Manual Dependency Injection using constructors
* Interface-based design
* Loose coupling between classes
* Multiple engine implementations
* Different car types
* Simple object creation and dependency wiring
* Clean package structure following Java best practices

## ▶️ How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA (or your preferred Java IDE).
3. Build the project using Maven.
4. Run the `Main` class.
5. Observe how different engine implementations are injected into different car objects.
